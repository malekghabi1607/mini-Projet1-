# mini-Projet1 — TP1 Science des Données (Titanic)

Ce dépôt contient le travail du **TP1 de Science des Données** basé sur le dataset **Titanic** (891 passagers).
Objectif : explorer les données, visualiser certaines distributions, analyser des facteurs liés à la survie
(sexe, âge, port d’embarquement, classe), gérer des valeurs manquantes, et produire des exports.

## Contenu du dépôt

- `TP1.py` : script Python du TP (avec commentaires/réponses).
- `Donnees/titanic.csv` : dataset Titanic.
- (optionnel) `Donnees/titanic.json` : export JSON.
- (optionnel) `Donnees/titanic.xml` : export XML.
- `TP1-6.pdf` : énoncé / support de cours.

## Prérequis

- Python 3.x

  - `pandas`
  - `numpy`
  - `matplotlib`
